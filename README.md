# TJSON for Ruby

A Ruby implementation of TJSON: Tagged JSON with Rich Types.

https://www.tjson.org

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tjson'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tjson
